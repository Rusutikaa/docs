CocoaError

Describes errors within the Cocoa error domain. Cocoa内のエラードメインを記述します。

Availability 有効性

  • iOS 7.0+
  • macOS 10.9+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 8.0+
