A value type for a string with associated attributes for portions of its text.

Overview 概要

Attributed strings are character strings that have attributes for individual characters or ranges of characters. Attributes provide traits like visual styles for display, accessibilty for guided access, and hyperlink data for linking between data sources. Attribute keys provide the name and value type of each attribute. System frameworks like Foundation and SwiftUI define common keys, and you can define your own in custom extensions.

Attributed strings store their attributes in attribute containers, and provide key path-based access by implementing dynamic member lookup. As a result, you can access an attribute by name with Swift’s dot syntax. For example, the following code sets the inlinePresentationIntent attribute to the value stronglyEmphasized, which a UI toolkit like SwiftUI can then render as bold text.

Attributes can also apply to only part of an attributed string, by applying the attribute to a range.

You can access portions of the string with unique combinations of attributes by iterating over the string’s runs property.

Markdown Support

You can create an attributed string by passing a standard String or Data instance that contains the Markdown syntax to initializers like init(markdown:options:baseURL:). The attributed string creates attributes according to the markup in the string.

Localized strings that you load from strings files with initializers like init(localized:options:table:bundle:locale:comment:) can also contain Markdown to add styling attributes. In addition, these localized attributed string initializers can apply the replacementIndex attribute, which allows you to determine the range of replacement strings, whose order may vary between languages. Localized attributed strings can also use the Markdown extension syntax to indicate parts of the string where the system can apply the automatic grammar agreement feature.

Attribute Scopes

The AttributedString API defines keys for common uses, such as text styling, semantically marking up formattable types like dates and numbers, and hyperlinking. You can find these in the AttributeScopes enumeration, which contains attributes for Foundation, SwiftUI, UIKit, and AppKit.

You can define your own attributes by implementing AttributedStringKey, and reference them by name by collecting them in an AttributeScope.
