URLSession.DataTaskPublisher

A publisher that delivers the results of performing URL session data tasks. あるパブリッシャー、それはURLセッションデータタスクを実行する結果を配達します。

Availability 有効性

  • iOS 7.0+
  • iPadOS 7.0+
  • macOS 10.9+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 11.0+

Declaring Publisher Topography パブリッシャーがよって立つ所を宣言する

typealias URLSession.DataTaskPublisher.Output
The kind of values published by this publisher. このパブリッシャーによって出版される値の種類。
typealias URLSession.DataTaskPublisher.Failure
The kind of errors this publisher might publish. このパブリッシャーが出版するかもしれないエラーの種類。

Creating a Data Task Publisher あるデータタスクパブリッシャーを作成する

init(request: URLRequest, session: URLSession)
Creates a data task publisher from the provided URL request and URL session. あるデータタスクパブリッシャーをこの提供されたURLリクエストとURLセッションから作成します。

Inspecting Data Task Properties データタスクプロパティを検査する

let request: URLRequest
The URL request performed by the data task associated with this publisher. このパブリッシャーと結びつけられたデータタスクによって実行されたURLリクエスト。
let session: URLSession
The URL session that performs the data task associated with this publisher. このパブリッシャーと結びつけられたデータタスクを実行するURLセッション。
