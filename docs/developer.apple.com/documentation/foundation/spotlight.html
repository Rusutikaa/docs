Spotlight

Search for files and other items on the local device, and index your app's content for searching. ローカルデバイス上のファイルや他の項目を検索します、そしてあなたのアプリのもつ内容を検索のために索引化します。

Technology

  • Foundation ファウンデーション

On This Page この項には

  • Topics 
  • See Also  参照 

Topics 話題

Queries 照会

class NSMetadataQuery
A query that you perform against Spotlight metadata. あなたがSpotlightメタデータに対して実行する照会。
protocol NSMetadataQueryDelegate
An interface that enables the delegate of a metadata query to provide substitute results or attributes. メタデータ照会の委任先に代理の結果や属性の提供を可能にするインターフェイス。

Items

class NSMetadataItem
The metadata associated with a file. あるファイルと結び付けられるメタデータ。

Files and Data Persistence ファイルとデータ永続

File System ファイルシステム
Create, read, write, and examine files and folders in the file system. ファイルシステムにおいてファイルとフォルダの作成、読み込み、書き出し、そして調査をします。
Archives and Serialization アーカイブとシリアライゼーション
Convert objects and values to and from property list, JSON, and other flat binary representations. オブジェクトと値をプロパティリスト、JSON、そして平坦なバイナリ表現へとおよびそれから変換します。
Preferences 環境設定
Persistently store domain-scoped pieces of information for configuring your app. あなたのアプリを設定するために情報のドメインスコープ断片を永続的に格納します。
iCloud
Manage files and key-value data that automatically synchronize among a user's iCloud devices. ユーザ所有のiCloudデバイス間で自動的に同期するファイルやキー値データを管理します。
Optimizing Your App’s Data for iCloud Backup
Minimize the space and time that backups take to create by excluding purgeable and nonpurgeable data from backups.
