View Layout and Presentation ビューのレイアウトと提示

Combine views in stacks, generate groups and lists of views dynamically, and define view presentations and hierarchy. ビューをスタックにおいて組み合わせて、ビューのグループとリストを動的に生成して、そしてビューの提示と階層を定義してください。

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Overview 
  • Topics 
  • See Also 

Overview 概要

Use stacks and lists to lay out the views of your user interface. You can combine static views with views that are dynamically generated from collections of data. All container views update and adjust the positions of their children in response to changes in content or interface dimensions. スタックとリストを使って、あなたのユーザインターフェイスのビューそれらを配置してください。あなたは、静的ビューを、動的にデータのコレクションから生成されるビューと結合できます。全てのコンテナビューは、内容またはインターフェイス次元それらにおける変化に答えて、それらの子の位置を更新および調節します。

Stacks

Building Layouts with Stack Views
Compose complex layouts from primitive container views.
struct HStack
A view that arranges its children in a horizontal line. あるビュー、それは、それの子らをある水平線に整列します。
struct VStack
A view that arranges its children in a vertical line. あるビュー、それは、それの子らをある垂直線に整列します。
struct ZStack
A view that overlays its children, aligning them in both axes. あるビュー、それは、それの子らを上に重ねます、それらを両方の軸で整列しています。

struct LazyVStack

Hierarchical Views 階層ビュー

struct NavigationView
A view for presenting a stack of views that represents a visible path in a navigation hierarchy.
struct NavigationLink
A view that controls a navigation presentation. あるビュー、それはナビゲーション提示を制御します。
struct OutlineGroup
A structure that computes views and disclosure groups on demand from an underlying collection of tree-structured, identified data. ある構造体、それはビューおよび開示グループを要求に応じてツリー構造の、識別可能なデータからなるある基礎をなすコレクションから計算します。
struct DisclosureGroup
A view that shows or hides another content view, based on the state of a disclosure control. あるビュー、それは別のコンテンツビューを、ある開示コントロールの状態に基づき表示するまたは隠します。
struct TabView
A view that switches between multiple child views using interactive user interface elements. あるビュー、それは複数の子ビューの間を切り替えます、対話式ユーザインターフェイス要素を使います。
struct HSplitView
A layout container that arranges its children in a horizontal line and allows the user to resize them using dividers placed between them. あるレイアウトコンテナ、それは、それの子らをある水平線に整列します、そしてそれらの間に置かれる仕切りを使ってユーザにそれらの大きさ変更を可能します。
struct VSplitView
A layout container that arranges its children in a vertical line and allows the user to resize them using dividers placed between them. あるレイアウトコンテナ、それは、それの子らをある垂直線に整列します、そしてそれらの間に置かれる仕切りを使ってユーザにそれらの大きさ変更を可能します。
