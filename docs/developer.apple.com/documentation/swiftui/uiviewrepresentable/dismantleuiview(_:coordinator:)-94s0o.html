Type Method 型メソッド

dismantleUIView(_:coordinator:)

Cleans up the presented UIKit view (and coordinator) in anticipation of their removal. 提示されたUIKitビュー（そしてコーディネータ）をそれらの除去を見越してきれいに片付けます。

Availability 有効性

  • iOS 13.0+
  • Mac Catalyst 13.0+
  • tvOS 13.0+
  • watchOS 6.0+

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Declaration 
  • Parameters 
  • Discussion 

Declaration 宣言 

static func dismantleUIView(_ uiView: Self.UIViewType, coordinator: Self.Coordinator)

Parameters パラメータ

uiView

Your custom view object. あなたのあつらえのビューオブジェクト。

coordinator

The custom coordinator instance you use to communicate changes back to SwiftUI. If you do not use a custom coordinator, the system provides a default instance. あなたが変更を逆にSwiftUIへと伝えるために使用するあつらえのコーディネータインスタンス。あなたがあつらえのコーディネータを使わないならば、システムは省略時のインスタンスを提供します。

Discussion 議論

Use this method to perform additional clean-up work related to your custom view. For example, you might use this method to remove observers or update other parts of your SwiftUI interface. このメソッドを使って、あなたのあつらえのビューに関連する追加のクリーンアップ作業を実行してください。例えば、あなたはこのメソッドを使って、オブザーバを取り除いたり、あなたのSwiftUIインターフェイスの他の部分を更新したりするかもしれません。
