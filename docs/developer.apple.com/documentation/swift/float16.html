Float16

A half-precision (16b), floating-point value type. 半精度（16b）、浮動小数点値型。

Availability

  • iOS 14.0+
  • iPadOS 14.0+
  • macOS 11.0+
  • Mac Catalyst 14.5+
  • tvOS 14.0+
  • watchOS 7.0+
  • Xcode 12.0+
