LazyFilterSequence

A sequence whose elements consist of the elements of some base sequence that also satisfy a given predicate. あるシーケンス、それは何らかの基盤シーケンスの要素でさらにまた与えられた述部を満足させるものから成る要素たちです。

Availability

  • iOS 8.0+
  • macOS 10.10+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 7.0+

Declaration 宣言 

@frozen struct LazyFilterSequence<Base> where Base : Sequence

Overview 概要

Note 注意

s.lazy.filter { ... }, for an arbitrary sequence s, is a LazyFilterSequence. s.lazy.filter { ... }は、随意のシーケンスsのための、ひとつのLazyFilterSequenceです。
