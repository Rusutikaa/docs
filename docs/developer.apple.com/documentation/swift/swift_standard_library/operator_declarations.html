If you declare a new operator without specifying a precedence group, it is a member of the DefaultPrecedence precedence group. DefaultPrecedence has no associativity and a precedence immediately higher than TernaryPrecedence. 中置演算子は、以下に優先順位の高い方から優先順位グループによって分類されます。あなたが新しい演算子を優先順位グループを指定せずに宣言するならば、それはDefaultPrecedence優先順位グループのメンバです。DefaultPrecedenceは、TernaryPrecedenceのすぐ上の結合性と優先順位を持ちません。

Programming Tasks プログラミング作業

Input and Output 入力と出力
Print values to the console, read from and write to text streams, and use command line arguments. 値をコンソールに出力し、テキストストリームから読み込みおよびそれへと書き出し、そしてコマンドライン引数を使います。
Debugging and Reflection デバッグとリフレクション
Fortify your code with runtime checks, and examine your values' runtime representation. あなたのコードを実行時チェックで強化し、そしてあなたの値の実行時表現を検査します。
Concurrency
Perform asynchronous and parallel operations.
Key-Path Expressions キーパス式
Use key-path expressions to access properties dynamically. キーパス式を使ってプロパティに動的にアクセスします。
Manual Memory Management 手動メモリ管理
Allocate and manage memory manually. メモリを手動でアロケートして管理します。
Type Casting and Existential Types 型キャストと存在型
Perform casts between types or represent values of any type. 型の間でキャストを実行したり、任意型の値を表します。
C Interoperability C互換性
Use imported C types or call C variadic functions. インポートされたC型を使ったり、C可変長引数関数を呼び出します。
