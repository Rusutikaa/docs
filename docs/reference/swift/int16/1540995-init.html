Initializer

init(bitPattern:)

Construct a Int16 having the same memory representation as the UInt16 bitPattern. No range or overflow checking occurs, and the resulting Int16 may not have the same numeric value as bitPattern–it is only guaranteed to use the same pattern of bits.
Int16をUInt16 bitPatternと同じメモリ表現を持つようにして組み立てます。範囲またはオーバーフロー検査が起こることはありません、そして結果のInt16はbitPatternと同じ数値を持たないでしょう - それはただ同じビットのパターンを使うことだけを保証されます。
